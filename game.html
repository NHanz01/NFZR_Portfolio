Nazomeita Not Expedea

「謎めいた(Nazomeita)」


謎めいた or translated as "Enigmatic" is a visual novel game created solely by me using Scratch back in Diploma for subject Basic Animation In Multimedia. The game starts with player wake up and find themselves trapped in a game. There are three mini games player needs to complete to return to reality. The first game requires interaction with 'W A S D' keys on keyboard to move, second game is a mouse clicking game and the final game requires player's device camera as it uses motion sensor.

「Not Expedea」


Not Expedea is a simple Spatial.io game developed using Unity engine, to demonstrates multiplayer game, for subject Real-Time Computer Graphics (SECV3123). The assets used in the game are free models, mostly taken from sketchfab, in .fbx type.
