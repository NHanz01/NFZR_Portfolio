「Real-Time Computer Graphics」

Compilation of assignments completed during the Real-Time Computer Graphics subject, consisting:

WebVR Hologram Marker AR

WebVR using Glitch.com


This assignment requires me to develop a simple WebVR application using A-Frame and it was developed using glitch.com. The character is animated using Mixamo and user can click on the objects to interact. (Update: Since web hosting is no longer supported on glitch, this WebVR was transferred to Github repository.)


Hologram MagicaVoxel


This assessment requires me to build a pyramid hologram using a clear/transparent sheet, build a 3D model/character using MagicaVoxel, animate them using Mixamo and finally generate a display for hologram using Unity. In the demo video, the first part shows the live demo of how the model would look like once projected on the hologram pyramid. A boy in white 'Baju Melayu' is dancing to a song called 'Sesuci Lebaran'. In the second part of the video, anyone can try to display the boy by placing a pyramid hologram on top of the device screen.


Marker-based AR Vuforia


This assessment requires me to display animated 3D objects on top of 5 different markers using Vuforia and Unity. Then, the animations would change once the markers collide with each other.
