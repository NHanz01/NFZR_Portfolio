「Web Programming: AnimeXhibition」

A website built for Multimedia Web Programing Mini Project; Digital Communities - RIA (Rich Internet Application) online platform that brings together people with shared interests. The project implements JQuery, AJAX and Jikan.moe API.

Live Website

About Project


ArtXhibition is an online RIA (Rich Internet Application) platform that introduces Anime and Art to the communities. Through the website, Anime lovers can gather to check out the latest Anime recommendations, find out details on Anime characters, try guessing the Anime characters, and check out the posters related to Anime. It also aimed to help newbies to discover about anime, including where to find anime events, cosplay stores and other recommended sites.

AnimeXhibition integrates Jikan.moe API that utiilizes MyAnimeList (MAL), a well-known anime website's database to display the data. The site was built by four groupmates, including myself, using Tooplate template and hosted on GitHub for convenient version control.
